## 2. 사용 기술

  - Java
  - Spring Boot
  - MySQL 8.5.26
  - Python 3.9
  - Tomcat
  - Mybatis
#### `Front-end`
  - React Native
  - Javascript
  - Axios
#### `API`
  - Kakao Map API
  - PortOne API
#### `Model`
  - LSTM
  - KoELECTRA
  - Fast API
